# Getting Started

## Introduce
**Jordy**은 타입스크립트를 기반으로 제작된 프론트엔드 핵심 업무 구성을 위한 툴킷입니다.

해당 툴킷은 여러가지 기능을 내포합니다.

- network
- form validation
- storage manager
- route system
- redux query
- cache
- etc ..


## Installation

### NPM

```shell
npm install jordy
```

### YARN

```shell
yarn add jordy
```
